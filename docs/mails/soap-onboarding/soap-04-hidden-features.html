Guten Tag,

ein paar Features von engpassradar.ch, die wenig bekannt sind — aber täglich nützlich:

01 — RSS-Feeds pro Wirkstoffgruppe

Jede ATC-Gruppe hat einen eigenen RSS-Feed. Engpässe für Ihre relevanten Wirkstoffe direkt in Ihr PMS oder Monitoring-System einbinden:
engpassradar.ch/wirkstoff/[ATC]/feed.xml

02 — ATC-spezifische E-Mail-Alerts

Auf jeder Wirkstoff-Seite können Sie einen kostenlosen Alert einrichten — Sie werden benachrichtigt, sobald sich die Lage für diesen Wirkstoff ändert.

03 — CSV-Export

Der tagesaktuelle Datensatz lässt sich direkt aus dem Dashboard als CSV exportieren — mit allen Filtern angewendet.

04 — Öffentliche REST-API

Alle Endpunkte sind ohne Registrierung nutzbar (100 Anfragen/Stunde). Für produktive Integrationen gibt es Engpassradar Pro mit 10'000 Anfragen/Tag.
